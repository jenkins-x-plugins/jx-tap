### Linux

```shell
curl -L https://github.com/jenkins-x-plugins/jx-tap/releases/download/v{{.Version}}/jx-tap-linux-amd64.tar.gz | tar xzv 
sudo mv jx-tap /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/jenkins-x-plugins/jx-tap/releases/download/v{{.Version}}/jx-tap-darwin-amd64.tar.gz | tar xzv
sudo mv jx-tap /usr/local/bin
```
